В директории playbooks можно найти роли для установки сервисов, которые требовались для проекта. Если каких-то ролей не хватает, значит эти сервисы были установлены вручную по какой-либо причине (блокирование российских ip, графический установщик или еще что-то). Конфиги правились в течение реализации всего проекта, поэтому сюда добавлены не были, так как это не имеет смысла.
